Better New Tabs – Privacy Policy

Last updated: December 2025

Better New Tabs is a Chrome extension that replaces the default new tab page with a customizable dashboard. This policy explains what information the extension uses, how it is stored, and how your privacy is protected.

Information We Collect

Better New Tabs does not collect, transmit, sell, or share any personal data with external servers or third parties.

The extension stores a small amount of information locally on your device, using Chrome’s built-in storage API. This information never leaves your browser.

Data stored locally includes:

  • Your name – required to display a personalized greeting on the new tab page.
  • Layout and appearance settings – dashboard layout, theme selection, and customization options.

No other personal information is collected or stored.

Bookmarks Access

Better New Tabs uses Chrome’s bookmarks API for the following:

  • Reading and displaying your existing bookmarks
  • Supporting bookmark search
  • Allowing drag-and-drop rearrangement of bookmarks
  • Allowing bookmark editing

Bookmark data is accessed only to display and manage it inside the new tab page. Bookmark data is never collected, transmitted, synced externally, or shared.

Favicon Usage

Better New Tabs displays website icons (favicons) for bookmarks using Chrome’s built-in favicon service.

  • These icons are retrieved locally by the browser.
  • No external requests are made by the extension to third-party servers for favicons.
  • Favicon usage is purely visual and remains inside the browser.

Use of Chrome Storage

The extension uses Chrome’s storage API exclusively to store:

  • Your name (for the personalized greeting)
  • Dashboard layout and UI preferences
  • Customization settings

All data saved through Chrome storage remains entirely on your device. No data is uploaded, transmitted, synced externally, or shared with third parties.

No External Servers, Tracking, or Remote Code

Better New Tabs does not:

  • Send any data to external servers
  • Use analytics or tracking technologies
  • Use cookies or external storage mechanisms
  • Load or execute remote scripts or remote code
  • Communicate with any external domains for user data

All processing happens locally within the extension in your browser.

Data Sharing

Better New Tabs:

  • Does not share data
  • Does not sell data
  • Does not transfer data to any entity
  • Does not use data for advertising, profiling, or marketing

Your information stays completely inside your browser.

Children’s Privacy

Better New Tabs does not collect personal data and contains no content that is inappropriate for children. It is safe for users of all ages.

Security

Because all data is stored locally within Chrome:

  • No data is transmitted externally
  • No external databases are used
  • Security risks associated with data transfer are minimized

Your information remains fully under your control at all times.

Changes to This Policy

If this privacy policy is updated, the new version will be posted on the official policy page. Users are encouraged to review the policy periodically for any changes.
